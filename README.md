# 用法
## 1. 添加依赖
克隆本项目添加依赖或者在Gradle中添加依赖:
```gradle
    compile 'com.huayuxun.whale.whaleradargraph.widget.WhaleShareView'
```
 > Gradle方式可能暂时无法使用

## 2. 布局文件中定义
```xml  
    <com.huayuxun.whale.WhaleShareView
        android:id="@+id/whaleLineChart"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content" />
```

## 3. 添加数据
```java
        whaleShareView = (WhaleShareView) findViewById(R.id.whaleShareView);
        
```
